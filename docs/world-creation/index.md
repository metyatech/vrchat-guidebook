# ワールドの作り方

<div class="meta-badges">
  <span class="meta-badge">対象: 初級-中級</span>
  <span class="meta-badge">目安: 2-4時間</span>
  <span class="meta-badge">難易度: Normal</span>
</div>

<div class="section-block">
<h2>この章でできるようになること</h2>

- ワールド制作の設計から公開までを一通り実施できる
- 初期段階で最適化ポイントを押さえた構成にできる
- 公開前チェックを使って品質を安定させられる

</div>

<div class="section-block">
<h2>基本フロー</h2>

<ol class="step-list">
  <li>目的と導線を決めてシーン構成を作る</li>
  <li>ギミック実装とパフォーマンス最適化を進める</li>
  <li>公開設定を整えてアップロードし、実地確認する</li>
</ol>

</div>

## まず覚えること

- シーン構成と導線づくり
- パフォーマンス最適化
- アップロード設定と公開手順

## 公開前チェック

<ul class="checklist">
  <li>PC/Quest 対応方針を事前に決めている</li>
  <li>描画負荷の高い要素をプロファイルで確認した</li>
  <li>スポーン地点、導線、説明文が分かりやすい</li>
</ul>

## 運用メモ

- 公開後はバージョン履歴と変更点を記録する
- ユーザー導線は実際の利用ログを見て改善する
