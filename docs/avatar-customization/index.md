# 改変のやり方

<div class="meta-badges">
  <span class="meta-badge">対象: 初級-中級</span>
  <span class="meta-badge">目安: 1-2時間</span>
  <span class="meta-badge">難易度: Normal</span>
</div>

<div class="section-block">
<h2>この章でできるようになること</h2>

- 改変に必要な環境を正しく準備できる
- アバター導入からアップロードまでの流れを理解できる
- 初期トラブル時に確認すべきポイントが分かる

</div>

<div class="section-block">
<h2>基本フロー</h2>

<ol class="step-list">
  <li>Unity と SDK のバージョンを固定して環境構築</li>
  <li>アバター導入と必要コンポーネントのセットアップ</li>
  <li>パフォーマンス確認後にアップロード</li>
</ol>

</div>

## まず覚えること

- Unity プロジェクト準備
- アバターの導入
- コンポーネント設定
- アップロード前のチェック

## つまずきやすいポイント

- SDK のバージョン違いによるエラー
- シェーダーやマテリアル互換の問題
- ポリゴン数・PhysBone の過剰設定

## チェックリスト

<ul class="checklist">
  <li>Console にエラーが残っていない</li>
  <li>Expression/FX の設定が意図どおり動く</li>
  <li>アップロード後の見た目と挙動を実機で確認した</li>
</ul>

> 次は `ワールドの作り方` へ進み、制作と公開の全体像をつかみましょう。
